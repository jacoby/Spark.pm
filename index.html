Installation

Spark.pm is a Perl library, adapted from Zack Holman's Spark bash script.

Usage

The quick and easy way to handle it is to put it in the ~/bin directory along with your spark-making Perl script. Second choice is to put it in your ~/lib directory, and add use lib '/home/you/lib' before including it.

Here's my minimal spark.pl program.

#!/usr/bin/perl
use feature 'say'; # feature detection trumps version detection any day!
use strict ;
use warnings ;
use utf8 ; # the sparks are unicode
use lib '/home/jacoby/lib' ;
use Spark ;

say spark @ARGV ;
